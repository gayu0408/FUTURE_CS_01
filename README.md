Internship

Future Interns - Cyber Security Internship

Task 1- Vulnerability Assessment Report for a Live Website

Objective:
The objective of this task was to perform a vulnerability assessment on a live web application using industry-standard security tools and identify potential security weaknesses.

Target Website- http://demo.testfire.net

Tools Used:
* Nmap
* OWASP ZAP
* Google Chrome DevTools

Methodology:
1. Reconnaissance- Performed network scanning using Nmap to identify open ports and exposed services.

2. Vulnerability Assessment- Used OWASP ZAP to discover web application vulnerabilities and security misconfigurations.

3. Analysis- Classified findings according to their risk level and documented remediation recommendations.

Key Findings:

High Risk-
* SQL Injection
* Cross Site Scripting (XSS)

Medium Risk-
* Absence of Anti-CSRF Tokens

Low Risk-
* Missing Content Security Policy Header
* Missing Anti-Clickjacking Header
* Cookie without SameSite Attribute
* Information Disclosure
* Server Information Exposure

Deliverables-
* Vulnerability Assessment Report
* Nmap Scan Results
* OWASP ZAP Scan Report
* Supporting Screenshots
